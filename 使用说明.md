# 电厂运行人员工作笔记系统 - 跨平台便携版

## 🚀 快速启动

### Windows
1. **双击 `启动.bat`** → 自动检测Python → 安装依赖 → 启动服务 → 自动打开浏览器
2. 浏览器自动打开 `http://localhost:5000`
3. 用完后关闭命令行窗口，或双击 `停止.bat`

### macOS / Linux
1. **终端运行 `./start.sh`** → 自动检测Python → 安装依赖 → 启动服务 → 自动打开浏览器
2. 浏览器自动打开 `http://localhost:5000`
3. 按 `Ctrl+C` 停止服务，或另开终端运行 `./stop.sh`

---

## 💾 U盘便携运行

### Windows（无需安装Python）
1. 双击 **`setup_portable.bat`** → 自动下载嵌入版Python（约8MB）+ 依赖
2. 配置完成后，整个文件夹拷贝到U盘即可
3. 在任意Windows电脑上双击 **`启动.bat`** 即可运行

### macOS / Linux（使用虚拟环境）
1. 终端运行 **`./setup_mac.sh`** → 自动用系统Python创建venv + 安装依赖
2. 配置完成后，整个文件夹拷贝到U盘即可
3. 在其他Mac上运行 **`./start.sh`** 即可运行
   - 如果目标Mac架构不同（Intel vs Apple Silicon），需重新运行 `./setup_mac.sh`

## 📁 文件结构

```
├── 启动.bat              ← Windows启动（主入口）
├── 停止.bat              ← Windows停止服务
├── setup_portable.bat    ← Windows一次性配置便携Python
├── start.sh              ← macOS/Linux启动（主入口）
├── stop.sh               ← macOS/Linux停止服务
├── setup_mac.sh          ← macOS/Linux一次性配置虚拟环境
├── 使用说明.md            ← 本文件
├── API文档.md             ← REST API 文档（命令行/大模型上传笔记）
├── note_cli.py            ← 命令行工具（通过 API Token 增删查笔记）
├── requirements.txt      ← 运行时依赖（已锁定版本范围）
├── requirements-dev.txt  ← 开发/测试依赖（pytest）
├── 电厂运行人员工作笔记模板.xlsx  ← Excel模板
├── backend/
│   ├── app.py            ← Flask后端（跨平台）
│   └── notes.db          ← SQLite数据库（自动生成）
├── frontend/
│   ├── index.html         ← Web前端
│   └── vendor/            ← 本地内置的前端库（marked / DOMPurify / SheetJS，离线可用）
├── tests/                ← 后端 API 测试（pytest）
├── python/               ← Windows便携Python（setup_portable.bat生成）
└── venv/                 ← macOS/Linux虚拟环境（setup_mac.sh生成）
```

## 🔧 功能

- 5大领域24子类知识体系（A01-E04编码）
- 笔记增删改查 + 关键词搜索
- 按等级/来源/领域/章节筛选
- 从Excel导入笔记
- Agent批量追加接口
- **API Token 认证 + 命令行/大模型批量上传**（详见 `API文档.md`）
- 安全加固：笔记内容渲染消毒、字段校验、登录失败限流、默认口令提醒
- 完全离线可用：Markdown / 消毒 / Excel 解析库均本地内置（`frontend/vendor/`），无需联网
- 跨平台运行（Windows / macOS / Linux）

---

## 🔒 安全提示

- 首次运行会自动创建管理员账号 **admin / admin123**。登录后页面顶部会出现提醒横幅，请尽快通过 🔑「修改密码」更换默认口令。
- 连续多次登录失败会触发临时锁定（默认 5 次/5 分钟），用于防止口令暴力破解。

## 🤖 命令行 / 大模型上传笔记

除网页端外，可通过 API 在命令行或大模型流水线中直接录入笔记，无需打开浏览器：

1. 用管理员账号登录后调用 `POST /api/tokens` 创建一个 API 令牌（或部署时设置环境变量 `NOTES_API_TOKEN`）。
2. 使用随附的命令行工具：

   ```bash
   export NOTES_API_URL=http://localhost:5000
   export NOTES_API_TOKEN=ntk_你的令牌
   python note_cli.py sections          # 查看章节编码
   python note_cli.py import notes.json # 批量上传大模型整理好的笔记
   ```

完整接口说明、字段规范与「给大模型的提示词模板」见 **`API文档.md`**。

## ⚠️ 注意事项

- **数据库文件 `notes.db`** 是跨平台兼容的（SQLite格式），Windows和Mac共用同一份数据
- **U盘文件系统**建议使用 exFAT 格式（Windows和macOS都可读写）
- 如果U盘使用 NTFS 格式，macOS默认只能读取，需安装第三方驱动才能写入
- Windows便携Python约占用 30MB 空间，macOS虚拟环境约占用 50MB
