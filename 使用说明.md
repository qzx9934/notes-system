# 电厂运行人员工作笔记系统

基于 Flask + SQLite 的 Web 笔记系统，已部署在云服务器，浏览器（含手机）即可访问使用。

## 🚀 快速开始

1. 打开部署好的站点地址（你的域名 / 公网 IP，HTTPS）。
2. 首次使用默认管理员账号 **admin / admin123** 登录。
3. 登录后页面顶部会出现提醒横幅，**立即通过 🔑「修改密码」更换默认口令**。
4. 手机端可「添加到主屏幕」安装为 App（PWA），像原生应用一样全屏使用。

> 生产部署 / 更新 / 排错请见 [`docs/deploy/宝塔部署.md`](docs/deploy/宝塔部署.md)。

---

## 🖥️ 本地运行（开发 / 预览）

需要本机已安装 Python 3：

```bash
python3 -m venv venv
venv/bin/python -m pip install -r requirements.txt
venv/bin/python backend/app.py        # 启动后访问 http://localhost:5000
```

> Windows：`python -m venv venv` → `venv\Scripts\python -m pip install -r requirements.txt` → `venv\Scripts\python backend\app.py`。
> 生产环境请用 gunicorn 跑 `wsgi:app`（见部署文档），不要用开发服务器对外。

## 🔧 功能

- 5大领域26子类知识体系（A01-E04编码）
- 笔记增删改查 + 关键词搜索（SQLite FTS5 全文索引，支持中文，自动回退 LIKE）
- **笔记内容支持插入图片**：编辑器「🖼」按钮选图，或直接粘贴截图 / 拖拽图片自动上传（仅管理员，JPG/PNG/GIF/WEBP，默认≤5MB）；删除/编辑笔记会自动回收无引用的孤儿图片，也可在「🎫 令牌管理」面板手动清理
- 按等级/来源/领域/章节筛选
- 从Excel导入笔记
- Agent批量追加接口
- **API Token 认证 + 命令行/大模型批量上传**（详见 `API文档.md`）
- **资料整理工作流**：把文本/Word/PDF/PPTX 整理成规范笔记并上传（详见 `docs/资料整理工作流程.md`）
- 安全加固：笔记内容渲染消毒、字段校验、登录失败限流、默认口令提醒
- 前端依赖本地内置（`frontend/vendor/`），无外部 CDN 依赖
- **PWA 支持**：手机浏览器「添加到主屏幕」即可像 App 一样全屏使用

## 🔒 安全提示

- 首次运行会自动创建管理员账号 **admin / admin123**。登录后页面顶部会出现提醒横幅，请尽快通过 🔑「修改密码」更换默认口令。
- 连续多次登录失败会触发临时锁定（默认 5 次/5 分钟），用于防止口令暴力破解。

## 🌐 云服务器 / 公网部署加固

部署到公网（有域名/公网 IP）时，请通过环境变量开启以下加固项：

| 环境变量 | 作用 | 公网推荐值 |
| --- | --- | --- |
| `NOTES_HTTPS` | 站点经 HTTPS 提供时设为 `1`：下发 `Secure` Cookie + HSTS 头 | `1`（**务必先配好 HTTPS 证书**） |
| `NOTES_TRUST_PROXY` | 位于 Nginx/Caddy 等反向代理之后时设为 `1`：按 `X-Forwarded-*` 还原真实客户端 IP（登录限流依赖它） | `1` |
| `NOTES_CORS_ORIGINS` | 允许跨域访问的来源（逗号分隔）；不设则不开放跨域（同源前端无需 CORS） | 一般留空 |
| `NOTES_COOKIE_SAMESITE` | 会话 Cookie 的 SameSite 策略（防 CSRF） | 默认 `Lax`，可设 `Strict` |
| `NOTES_SECRET_KEY` | 会话签名密钥（不设会自动生成并持久化到 `.secret_key`） | 设为强随机串 |
| `NOTES_API_TOKEN` | 主 API 令牌（命令行/大模型上传用） | 按需设置 |
| `NOTES_UPLOAD_DIR` | 笔记图片存储目录 | 默认 `backend/uploads/`，**需与 `notes.db` 一同持久化备份** |
| `NOTES_MAX_UPLOAD_MB` | 单张图片大小上限（MB） | 默认 `5` |
| `NOTES_DEEPSEEK_API_KEY` | DeepSeek 密钥，启用「AI 总结」功能必填（不设则该功能返回明确提示） | 你的 DeepSeek 密钥 |
| `NOTES_DEEPSEEK_MODEL` | AI 总结使用的模型 ID | 默认 `deepseek-v4-flash` |
| `NOTES_DEEPSEEK_BASE_URL` | DeepSeek 接口地址 | 默认 `https://api.deepseek.com` |
| `NOTES_DEEPSEEK_TIMEOUT` | AI 总结请求超时（秒） | 默认 `60` |

会话 Cookie 始终带 `HttpOnly`（禁 JS 读取）；并默认下发安全响应头（`X-Frame-Options: DENY`、`X-Content-Type-Options: nosniff`、`Referrer-Policy`）。

**systemd / gunicorn 示例：**

```ini
Environment=NOTES_HTTPS=1
Environment=NOTES_TRUST_PROXY=1
Environment=NOTES_SECRET_KEY=请替换为强随机串
ExecStart=/path/venv/bin/gunicorn -w 2 -b 127.0.0.1:5000 wsgi:app
```

> 反向代理（Nginx）需转发 `X-Forwarded-For` / `X-Forwarded-Proto`（`proxy_set_header` 默认即带），TLS 在代理层终止。完整步骤见 `docs/deploy/宝塔部署.md`。

**上线检查清单**：① 配好 HTTPS 证书 → ② 设 `NOTES_HTTPS=1`、`NOTES_TRUST_PROXY=1`、`NOTES_SECRET_KEY` → ③ 首次登录立即修改默认口令。

## 🤖 命令行 / 大模型上传笔记

除网页端外，可通过 API 在命令行或大模型流水线中直接录入笔记，无需打开浏览器：

1. 用管理员账号登录后调用 `POST /api/tokens` 创建一个 API 令牌（或部署时设置环境变量 `NOTES_API_TOKEN`）。
2. 使用随附的命令行工具：

   ```bash
   export NOTES_API_URL=https://你的域名
   export NOTES_API_TOKEN=ntk_你的令牌
   python note_cli.py sections          # 查看章节编码
   python note_cli.py import notes.json # 批量上传大模型整理好的笔记
   ```

完整接口说明、字段规范与「给大模型的提示词模板」见 **`API文档.md`**；把资料整理成笔记的完整流程见 **`docs/资料整理工作流程.md`**。

## ⚠️ 注意事项

- **数据库文件 `backend/notes.db`** 是全部用户数据所在，请在部署机上**定期备份**；代码更新（`git pull` + 重启）不会影响它。
- 首次部署后**务必修改默认口令** `admin/admin123`。
- 公网访问务必配置 HTTPS 并设置上表环境变量。
