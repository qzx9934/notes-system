🎯 系统简介

本系统是一个面向电厂运行人员的 Web 工作笔记管理平台，把零散的规程要点、操作经验、事故反馈、培训笔记，按统一的知识体系结构化沉淀下来，做到 随手记录、按需检索、AI 辅助整理。

  • 浏览器即可使用，电脑、手机均可（支持安装为手机 App）；
  • 内置 5 大领域 / 26 个子类 的电厂知识编码体系，每条笔记自动编号；
  • 支持全文搜索、多维筛选、图片、Markdown、收藏、卡片漫游；
  • 接入 DeepSeek 大模型，提供 AI 总结、AI 填充、AI 整理等智能辅助；
  • 三级角色权限 + 共建者审批流，既鼓励人人参与，又保证内容质量。

🔐 登录与角色权限

首次使用默认管理员账号 admin / admin123 登录，请立即通过「🔑 修改密码」更换默认口令。系统设有登录失败限流（默认 5 次/5 分钟），可防止口令暴力破解。

角色权限范围
管理员读写全部内容，增删改即时生效；管理用户、API 令牌、AI 提示词；审批共建者提交的变更。
共建者可提交 新增 / 编辑 / 删除，但不直接生效——会生成一条「变更申请」，待管理员审批通过后才执行。
查看者只读：可浏览、搜索、收藏，不能修改内容。

每条笔记都归属一个章节编码，编号形如 A01-001，由系统按章节自增生成：

📝 日常操作：搜索 / 筛选 / 编辑

检索

  • 关键词搜索：顶部搜索框输入关键字，匹配标题 / 内容 / 标签 / 编号。后台采用 SQLite FTS5 全文索引（支持中文子串、大小写不敏感），数据量大时依然很快。
  • 多维筛选：左侧 / 顶部可按领域、章节、等级、来源筛选，并支持排序（编号 / 更新时间 / 创建时间 / 随机）。

新增与编辑（管理员 / 共建者）

  1. 点击「➕ 新增笔记」或卡片上的编辑按钮，打开编辑窗口。
  2. 选择所属章节、等级，填写要点标题与内容详情（支持 Markdown），可填关键词标签与来源。
  3. 点「保存」。管理员即时生效；共建者则提交为变更申请，等待审批。
抽屉详情页中，管理员可就地快速修改等级、来源、日期、标签——点一下即可改，标签以"小药丸"形式增删，无需打开完整编辑窗口。

🖼️ 插入图片与 Markdown

  • 编辑器工具栏支持加粗、标题、列表、引用、表格、链接等 Markdown 快捷插入，并可「👁 预览」。
  • 插入图片三种方式：点「🖼」选图、直接粘贴截图、拖拽图片到编辑框，均会自动上传（JPG/PNG/GIF/WEBP，默认 ≤5MB）。
  • 删除或编辑笔记时，系统会自动回收无人引用的"孤儿图片"，节省存储。

⭐ 收藏卡片

每位用户拥有独立的收藏夹，互不影响：

  • 在列表行或卡片详情中点击 ☆ 即可收藏 / 取消；
  • 通过「我的收藏」筛选，可一键列出自己收藏的全部笔记，方便随时复习重点。

🎴 笔记漫游

「笔记漫游」以全屏卡片形式逐条翻阅笔记，适合碎片时间随机复习。卡片可翻面查看 AI 总结，并可直接收藏当前卡片。

🤖 AI 智能功能

系统接入 DeepSeek 大模型（密钥经环境变量配置，不写入代码）。以下功能共用同一套密钥 / 模型：

① AI 总结

对单条笔记，以"集控运行人员"视角把内容提炼成 Markdown 要点，保存于卡片背面，可重复生成覆盖。

② 一键（批量）总结

勾选多条笔记后一键批量生成总结（单批上限 50 条），默认跳过已有总结的；遇到限流会自动退避重试，并在结束时反馈成功 / 跳过 / 失败明细。

③ AI 填充

在编辑窗口填好正文后点「🪄 AI 填充」，大模型会根据正文自动推断并填入 标题 / 标签 / 章节 / 等级 / 来源，人工核对后保存即可。

④ AI 整理

点「🧹 AI 整理」对正文做轻度规整，并同时完成上面的字段填充（一次调用，节省 token）：

  • 删除排班、通知、"即日起"等与知识无关的事务性内容；
  • 正文小序号乱序 / 格式不统一时，按并列知识点规范重排为 1. 2. 3.；
  • 克制改动，不修改技术数值与术语，避免把内容改错；
  • 整理后可点「↩ 撤回整理」一键恢复整理前的正文。

⑤ 提示词可配置（管理员）

管理员可在「🎫 令牌管理」面板随时修改 AI 总结提示词 与 AI 整理提示词（两者独立），留空保存即恢复内置默认，改后立即生效。

AI 功能需部署时配置 NOTES_DEEPSEEK_API_KEY 环境变量，未配置时相关按钮会给出明确提示。

📥 共建者审批流程

为兼顾"人人可补充"与"内容质量可控"，共建者的所有写操作都走审批：

  1. 共建者新增 / 编辑 / 删除笔记 → 系统生成一条变更申请，原内容暂不改变。
  2. 管理员在「📥 变更申请」收件箱以列表查看待办（支持翻页），点任一条进入明细页，改动以左右并列方式展示（左原值、右新值，正文完整对照）。
  3. 管理员批准则改动真正执行；驳回可填写理由。若目标笔记已被删除等导致无法执行，会自动驳回提示。

⚙️ 管理：用户 / 登录日志 / 令牌

  • 👥 用户管理：新增用户并指定角色（管理员 / 共建者 / 查看者），重置密码、删除用户；列表显示每位用户的最后上线时间。
  • 登录日志：可查看各用户的历次登录记录（时间、IP），便于安全审计。
  • 🎫 API 令牌管理：创建 / 吊销用于命令行或大模型批量上传的令牌；同面板可配置 AI 提示词、清理孤儿图片、笔记查重。
  • 💬 反馈建议 / 📨 反馈箱：任何登录用户都可点击「💬 反馈建议」提交使用问题或改进建议；管理员在「📨 反馈箱」查看、标记已处理或删除，未处理数量会以角标提示。

📷 你上传的图片会在服务器端自动压缩（保清晰度、减体积），无需手动处理。

📱 手机安装（PWA）

用手机浏览器打开站点后，通过浏览器菜单选择「添加到主屏幕」，即可把系统安装为 App，像原生应用一样全屏使用，断网时也能打开外壳界面。

💻 命令行 / 大模型批量上传

除网页端外，可用 API 令牌在命令行或大模型流水线中批量录入笔记，无需打开浏览器：

export NOTES_API_URL=https://你的域名
export NOTES_API_TOKEN=ntk_你的令牌
python note_cli.py sections           # 查看章节编码
python note_cli.py import notes.json  # 批量上传整理好的笔记

完整接口、字段规范与"给大模型的提示词模板"见随附的 API 文档。推荐流程：大模型把零散资料整理成规范 JSON，再经 POST /api/notes/ingest 一次性上传。
